# Graph Scripts

Primary entrypoint:

- `node .codex/graph/scripts/trace-impact.mjs <command> ...`

Primary commands:

- `inspect`
- `impact`
- `sync-set`
- `plan-change`
- `validate`

Shared `.codex` change control:

1. run `plan-change` before editing
2. edit graph and files
3. run `validate` after editing
