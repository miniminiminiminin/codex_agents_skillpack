# Graph-Governed Editing Skill Rules

This directory owns the reusable skill for graph-first shared-asset editing.

## Scope

- Applies to `.codex/skills/graph-governed-editing/**`.
- Keep this skill narrow: graph lookup, impact tracing, sync set definition, edit discipline.

## Rules

- Do not absorb role behavior, release policy, or template field policy into this skill.
- Route outward to prompts, workflows, and templates through the graph.
- Keep the checklist short enough to execute mechanically.
